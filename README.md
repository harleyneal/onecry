# OneCry Pensacola

24/7/365 prayer initiative uniting churches across Pensacola, FL.

Built with Next.js, Tailwind CSS, and deployed on Vercel.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

- `src/app/page.tsx` — Main landing page with all sections
- `src/app/layout.tsx` — Root layout with metadata and SEO
- `src/app/globals.css` — Global styles and brand colors
- `public/assets/` — Brand logos and symbol SVGs
- `public/favicon.ico` — Favicon generated from symbol
